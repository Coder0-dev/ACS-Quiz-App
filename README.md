# Chemistry Quiz App

A minimalist, high-performance quiz application designed for chemistry students, featuring real-time LaTeX rendering for complex equations, randomized questions, and a "Visual Precision" aesthetic.

## Tech Stack
* **Framework:** React 18 + Vite (TypeScript)
* **Styling:** Tailwind CSS
* **Math Rendering:** `react-katex` (KaTeX)
* **Icons:** `lucide-react`

## Quick Start

1. **Initialize the project:**
   ```bash
   npm create vite@latest chemistry-quiz -- --template react-ts
   cd chemistry-quiz
